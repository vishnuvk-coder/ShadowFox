# ShadowFox AIML Internship Projects

This repository contains all tasks completed during the **ShadowFox AIML Internship**.

## Internship Tasks Completed

### 1. Image Tagging Project (Beginner Level)

An Image Classification project using **TensorFlow** and **CNN (Convolutional Neural Network)** to classify images into different categories.

**Classes Used:**
- Cat
- Dog
- Car

**Technologies Used:**
- Python
- TensorFlow
- NumPy
- Matplotlib

**Project Features:**
- Image Classification
- CNN Model
- Model Training and Prediction

---

### 2. Car Selling Price Prediction (Intermediate Level)

A **Machine Learning Regression Model** that predicts the approximate selling price of a car based on various features.

**Features Used:**
- Fuel Type
- Years of Service
- Showroom Price
- Number of Previous Owners
- Kilometers Driven
- Seller Type
- Transmission Type

**Technologies Used:**
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

**Project Features:**
- Car Price Prediction
- Data Preprocessing
- Model Training
- Selling Price Analysis

---

### 3. Sentiment Analysis using Hugging Face (Advanced Level)

An **AI-driven Natural Language Processing (NLP)** project using a **pre-trained Language Model (LM)** from **Hugging Face Transformers** to classify text sentiment.

**Sentiment Classes:**
- Positive
- Negative

**Technologies Used:**
- Python
- Hugging Face Transformers
- PyTorch
- Jupyter Notebook

**Project Features:**
- Sentiment Analysis
- NLP Language Model Implementation
- Text Classification
- Performance Analysis
