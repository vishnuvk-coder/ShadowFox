# Image Tagging Project

This project is an Image Classification Model using TensorFlow and CNN.

## Classes
- Cat
- Dog
- Car

## Dataset
│
├── cat/
│   ├── cat1.jpg
│   ├── cat2.jpg
│   ├── cat3.jpg
│   └── ...
│
├── dog/
│   ├── dog1.jpg
│   ├── dog2.jpg
│   ├── dog3.jpg
│   └── ...
│
└── car/
    ├── car1.jpg
    ├── car2.jpg
    ├── car3.jpg
    └── ...

## Technologies Used
- Python
- TensorFlow
- NumPy
- Matplotlib

## Features
- Image Classification
- CNN Model
- Data Augmentation
- Accuracy Evaluation

## How to Run

Install libraries:

pip install (tensorflow, matplotlib, numpy)

Run training:

python train.py

Run prediction:

python predict.py

## Output
The model predicts image categories such as:
- Cat
- Dog
- Car
